stahl
=====

A dependently typed Lisp with algebraic effects.

Goals
-----

-	[ ] Interpreter for macro-less, untyped language
-	[ ] Typechecking
-	[ ] Compile to WebAssembly
-	[ ] Macro support
-	[ ] Web framework (with heavy "cheating" as a runtime)
-	[ ] Compile to native code
-	[ ] Rewrite the world!
